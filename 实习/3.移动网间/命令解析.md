# 1.ping命令
```R
ping -m 100 -i Route-Aggregation3 -a 221.183.90.122 221.183.90.121
ping ipv6 -m 100 -i HundredGigE1/0/0/12 -a 2409:8080:0:3:18E2:1880:14:0 2409:8080:0:3:18e2:1880:14:1
```
### 参数解读

|   参数   | 参数说明                                                                                                                                              | 取值                               |
| :----: | :------------------------------------------------------------------------------------------------------------------------------------------------ | :------------------------------- |
| **-m** | 指定发送下一个ICMP请求报文的等待时间。<br>ping发送ICMP Echo Request报文后等待应答（reply），缺省等待500毫秒后发送下一个ICMP Echo Request报文。可以通过time参数配置发送时间间隔。在网络状况较差情况下，不建议此参数取值小于2000毫秒。 | 整数形式，取值范围是1～10000，单位是毫秒。缺省值是500。 |
| **-i** | 指定发送ICMP Echo Request报文的接口。<br>对于广播式链路（以太网），指定的目的地址将作为报文的下一跳地址。\|                                                                                 | 字符串形式，取值范围是1～63。                 |
|        |                                                                                                                                                   |                                  |

# 2.查看指定 100G 以太网接口上光模块的数字诊断监控 (DDM/DOM) 信息
```R
display transceiver diagnosis interface HundredGigE1/0/0/1
```

|命令分段|含义|详细解释|
|---|---|---|
|`display`|显示命令前缀|H3C 设备中用于查询信息的基础命令关键字，类似其他厂商的`show`命令|
|`transceiver`|光模块|指接口上安装的光收发模块（如 SFP28、QSFP28 等 100G 光模块）|
|`diagnosis`|诊断|启用数字诊断功能，读取光模块 DDM/DOM 参数，包括温度、电压、电流、收发功率等|
|`interface`|接口|指定要查询的接口类型和编号|
|`HundredGigE`|100G 以太网|接口类型，表示 100Gbps 速率的以太网接口，也可简写为`100GE`|
|`1/0/0/1`|接口编号|H3C 高端设备接口编号格式：**槽位 / 子卡 / 端口组 / 端口**（普通设备可能为槽位 / 端口），表示第 1 个槽位、第 0 个子卡、第 0 个端口组的第 1 个 100G 端口|
### 输出示例
```D

     Wave.error  Freq.error  Temp  Voltage  Bias  RX power  TX power
```

### 输出字段解读

| 参数               | 单位  | 说明              |                          |

| **Temperature**  | ℃   | 光模块内部工作温度       | 0~70℃                    |
| **Voltage**      | V   | 光模块供电电压         | 3.2~3.4V                 |
| **Bias Current** | mA  | 激光发射偏置电流        | 因模块类型而异                  |
| **Tx Power**     | dBm | 发射光功率           | 符合模块规格，如 LR4 为 - 5~0dBm  |

### 关键知识点：CRC 错误是什么？

CRC（Cyclic Redundancy Check）= **循环冗余校验**

- 数据传输时会做校验，**校验失败就会记录为 CRC 错误**
- **CRC 错误 = 物理链路传输失败**（直接导致丢包、业务卡顿、中断）
- 100G 接口出现 CRC，**100% 是物理层问题**（和配置无关）

### 总结

1. 这是一条**过滤型查询命令**，专门快速看接口 CRC 错误
2. **CRC 错误 = 物理链路故障**，是 100G 接口故障的核心判断依据
3. 配合光模块诊断命令，可 100% 定位光模块 / 光纤故障
